- **用途**：主知识库、日记、项目草稿
- **分类**：
  - 生活日记 - 情绪记录、日常思考
  - 科研项目 - 金丹工程、宇宙学研究
  - 道家中医哲学 - 修行笔记、经典研读
  - 赚钱计划 - 商业想法、投资策略
  - 闲鱼智能体代理 - AI副业项目
  - 重大挑战项目 - 债务、纠纷、法律问题
  - 问题集 - 工作问题、运营思考
  - 爱好计划 - 机甲小说、其他兴趣

### Apple 提醒事项 (remindctl)
- **用途**：任务提醒、习惯养成
- **关键提醒**：
  - 每日工作汇报（给林生，Excel形式）
  - 喝酒前告知老婆
  - 每月25-26号营销计划
  - 周年庆等重要日期

### Apple 日历 (Calendar)
**16个日历分类 - 反映PDCA闭环管理系统：**

| 日历 | 用途 | 性格洞察 |
|------|------|----------|
| **plan** | 计划制定 | 重视前瞻性规划 |
| **work** | 工作执行 | 执行力强 |
| **welldone** | 已完成事项 | 对"完成感"有强烈需求，需要正反馈 |
| **Improving** | 持续改进 | 永不满足，追求优化 |
| **important** | 重要事项 | 优先级清晰 |
| **private** | 私人事务 | 公私分明 |
| **communite** | 社区/社交 | 重视人际关系网络 |
| **debt** | 债务追踪 | 不回避问题，主动面对财务压力 |
| **Dao** | 道家修行/金丹工程 | 精神追求与物质并行 |
| **HK** | 香港事务 | 跨地域业务管理 |
| **China** | 中国大陆事务 | 多地域布局 |
| **豆包** | AI助手工作流 | 科技应用能力强 |
| **计划的提醒事项** | 系统提醒 | 依赖系统而非记忆 |
| **生日** | 重要日期 | 重视人情往来 |
| **中国大陆节假日** | 节假日安排 | 提前规划休息 |
| **Siri建议** | AI智能建议 | 愿意尝试新技术 |

**管理系统特征：**
- **PDCA闭环**：plan → work → welldone → Improving 形成完整循环
- **财务透明**：专门用debt日历追踪，不逃避
- **精神物质并重**：Dao日历与work日历并存
- **科技赋能**：豆包日历显示对AI工具的深度应用
- **跨地域管理**：HK + China 双区域管理

**使用建议：**
- 定时任务已配置为每天18:00读取当天日历
- 每周六22:00完整回顾一周日历
