# 八星数字性格分析工具

🔮 一个极简的 Web 应用，基于八星数字能量学，通过分析手机号码来解读性格特征和人际关系匹配度。

![版本](https://img.shields.io/badge/version-1.0.0-gold)
![技术栈](https://img.shields.io/badge/技术栈-HTML%2BCSS%2BJS-cyan)
![许可证](https://img.shields.io/badge/license-MIT-green)

## ✨ 功能特性

### 1. 单人分析功能
- 输入手机号码，分析最后8位数字能量分布
- 八星能量分布可视化图表（Chart.js 环形图）
- 性格画像描述
- 适合的职业方向推荐
- 人际关系建议

### 2. 双人匹配功能
- 输入两个手机号码进行对比分析
- 匹配度百分比计算（带动画效果）
- 互补性分析
- 合作建议
- 注意事项提醒

### 3. UI 设计
- **风格**：玄学古风 + 现代科技融合
- **配色**：
  - 深色背景（#1a1a2e 深蓝紫）
  - 金色强调（#ffd700）
  - 青色点缀（#00d9ff）
- **特性**：
  - 移动端优先响应式设计
  - 星空背景动画
  - 卡片悬停效果
  - 渐变按钮动画
  - 平滑滚动体验

## 🚀 快速开始

### 方式一：直接打开（推荐）
1. 下载项目文件
2. 双击 `index.html` 在浏览器中打开
3. 无需任何构建工具或服务器

### 方式二：本地服务器
```bash
cd ba-xing-number
# 使用 Python 3
python -m http.server 8080
# 或使用 Node.js
npx serve .
# 然后访问 http://localhost:8080
```

### 方式三：GitHub Pages 部署
1. Fork 本项目到您的 GitHub 账号
2. 进入仓库 Settings → Pages
3. Source 选择 Deploy from a branch
4. Branch 选择 main，folder 选择 / (root)
5. 保存后访问 `https://yourusername.github.io/ba-xing-number/`

## 📝 使用说明

### 单人分析
1. 在输入框中输入手机号码（如 138 1234 5678）
2. 点击「开始分析」按钮
3. 查看八星能量分布图和性格解读

### 双人匹配
1. 切换到「双人匹配」标签
2. 输入两个手机号码
3. 点击「开始匹配分析」
4. 查看匹配度分数和详细分析
