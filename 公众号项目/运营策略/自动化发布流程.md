# 疗愈师阿慶 - 自动化发布流程

## 流程概述

本文档描述了疗愈师阿慶公众号的自动化发布流程，包括内容创作、排版、发布、数据监控等环节。

---

## 一、内容创作流程

### 1.1 选题确定

**选题原则：**
- 符合四大内容类型：个人故事40%、疗愈工具30%、观念觉醒20%、问答互动10%
- 紧扣目标读者需求：中年危机、家庭关系、创业失败、情感婚姻
- 每篇长度1500-2000字

**选题来源：**
- 庆哥提供的真实经历素材
- 读者留言中的问题和故事
- 社会热点与疗愈主题的结合
- 季节性话题（如春节前的情绪管理）

### 1.2 文章撰写

**职责分工：**
- 虾哥：撰写初稿（根据选题和素材）
- 庆哥：审核修改（24小时内完成）

**撰写标准：**
- 标题：20-25字，吸引但不标题党
- 结构：故事→感悟→工具/方法
- 金句：每篇至少3句易分享的句子
- 语言：温和、真诚、不说教
- 结尾：开放式提问 + 互动工具引导

**时间安排：**
- 周一：完成周二推文初稿
- 周四：完成周五推文初稿
- 周六：完成周日推文初稿

### 1.3 审核修改

**审核要点：**
- 内容真实性：所有故事必须真实
- 观点准确性：疗愈理念符合"允许一切"核心
- 语言风格：温暖、真诚、不鸡汤
- 工具实用性：配套工具经过验证
- 引导清晰：结尾提问和引导关注自然

**修改流程：**
1. 虾哥提交初稿给庆哥
2. 庆哥审阅并提出修改意见
3. 虾哥根据意见修改
4. 庆哥确认最终版本
5. 虾哥进入排版环节

## 二、排版流程

### 2.1 排版标准

**统一格式：**

1. **标题**
   - 居中对齐
   - 字号18-20
   - 加粗

2. **副标题**
   - 居中对齐
   - 字号14-16
   - 灰色（#666666）

3. **引导语**
   - 斜体
   - 字号14
   - 独立成段

4. **正文**
   - 字号14-16
   - 行间距1.5-1.8
   - 每段不超过4行

5. **小标题**
   - 左对齐
   - 字号16
   - 加粗
   - 每篇文章3-5个小标题

6. **引用/金句**
   - 引用格式
   - 居中或缩进
   - 背景色浅灰（#F5F5F5）

7. **互动工具**
   - 独立模块
   - 清晰的下载指引
   - 配套工具说明

8. **结尾**
   - 开放式提问
   - 引导关注
   - 分享呼吁

### 2.2 排版工具

**工具库：**
- 微信公众号编辑器（基础排版）
- Markdown转换工具（快速排版）
- 统一排版模板（提高效率）

**排版模板：**

见附件：《疗愈师阿慶-排版模板.md》

### 2.3 排版检查清单

发布前检查：
- [ ] 标题格式正确
- [ ] 正文字号一致
- [ ] 段落间距合适
- [ ] 小标题编号正确
- [ ] 金句格式突出
- [ ] 互动工具链接有效
- [ ] 手机端预览无问题
- [ ] 无错别字
- [ ] 引导关注清晰
- [ ] 分享呼吁自然

## 三、定时发布

### 3.1 发布时间

**固定推送时间：**
- 周二 23:00（观念觉醒型）
- 周五 23:00（轻松治愈型）
- 周日 23:00（深度共鸣型）

**时间选择原因：**
- 夜深人静时，适合反思和疗愈
- 避开工作时间，减少信息干扰
- 便于睡前阅读，留存率高

### 3.2 定时发布设置

**操作步骤：**

1. **预览检查**
   - 手机端预览文章
   - 检查排版是否正常
   - 测试互动工具链接

2. **定时设置**
   - 在公众号后台选择"定时发送"
   - 设置发布日期和时间
   - 确认推送目标（全部粉丝）

3. **发布前最后检查**
   - 检查定时时间是否正确
   - 检查文章是否有紧急修改
   - 检查工具链接是否有效

4. **确认发布**
   - 点击"确定"
   - 系统自动在指定时间推送

### 3.3 发布监控

**发布后检查：**
- 确认文章成功推送
- 检查手机端接收正常
- 检查互动工具链接可访问
- 检查留言功能正常

## 四、互动管理

### 4.1 互动工具管理

**工具库：**

| 工具名称 | 类型 | 适用场景 | 获取方式 |
|---------|------|---------|---------|
| 情绪觉察日记 | PDF | 疗愈工具类 | 文末下载 |
| 呼吸冥想音频 | MP3 | 冥想练习类 | 文末下载 |
| 30天允许练习表 | PDF | 个人成长类 | 文末下载 |
| 50天疗愈挑战表 | PDF | 长期陪伴类 | 群内专属 |
| 一对一咨询 | 文字 | 深度疗愈 | 分享获取 |

**工具上传：**
1. 将工具文件上传到云存储（如百度网盘、腾讯云）
2. 生成分享链接
3. 在文章中插入下载链接
4. 定期检查链接有效性

**工具更新：**
- 根据读者反馈优化工具
- 定期更新工具内容
- 工具迭代时通知已领取用户

### 4.2 留言回复

**回复原则：**
- 24小时内回复
- 温暖、真诚、不敷衍
- 不说教、不鸡汤
- 可引用文章内容

**回复流程：**
1. 虾哥草拟回复（留言后2小时内）
2. 庆哥审阅（留言后4小时内）
3. 发布回复（留言后12-24小时内）

**精选留言：**
- 每周精选3-5条优质留言
- 制作"读者心声"内容
- 征得同意后分享到公众号

### 4.3 私信互动

**私信管理：**
- 24小时内回复私信
- 收集用户需求和故事
- 回答具体问题
- 邀请入群（合适的情况下）

**互动工具：**
- 自动回复设置（欢迎语、常见问题）
- 关键词自动回复（如"工具"、"课程"）
- 人工跟进重点用户

## 五、数据监控

### 5.1 监控指标

**核心指标：**
- 阅读量
- 分享量
- 留言量
- 新增关注数
- 取消关注数

**转化指标：**
- 互动工具下载量
- 分享后领取人数
- 群内新增人数

### 5.2 数据统计

**日报（虾哥负责）：**
- 昨日阅读量、分享量、留言量
- 新增关注、取消关注
- 互动工具下载量
- 优质留言摘要

**周报（虾哥负责）：**
- 本周数据汇总
- 周环比变化
- 爆款内容分析
- 读者反馈汇总

**月报（庆哥+虾哥讨论）：**
- 月度数据总结
- 内容效果分析
- 读者画像更新
- 下月优化方案

### 5.3 数据优化

**优化方向：**
- 高阅读高分享：分析原因，复制成功经验
- 高阅读低分享：优化引导，增加互动设计
- 低阅读高分享：优化标题，增加曝光
- 低阅读低分享：调整选题或内容质量

**A/B测试：**
- 标题测试（不同标题版本）
- 推送时间测试（不同时间段推送）
- 封面图测试（不同封面图效果）
- 互动工具测试（不同工具领取方式）

## 六、自动化工具

### 6.1 文章排版自动化

**工具：** Markdown转换器

**操作流程：**
1. 用Markdown撰写文章
2. 使用转换工具转换为微信公众号格式
3. 复制到公众号编辑器
4. 微调格式（标题、图片、排版）

**模板库：**
- 个人故事模板
- 疗愈工具模板
- 观念觉醒模板
- 问答互动模板

### 6.2 定时发布自动化

**工具：** 微信公众号后台定时发送功能

**操作流程：**
1. 提前编辑好文章
2. 设置定时发布时间
3. 系统自动推送
4. 发布后自动统计数据

### 6.3 数据监控自动化

**工具：** 微信公众号后台数据统计

**自动统计：**
- 每日数据自动更新
- 每周数据自动汇总
- 每月数据自动生成报告
- 爆款内容自动提醒

### 6.4 互动工具自动化

**工具：** 云盘+关键词自动回复

**操作流程：**
1. 工具文件上传云盘
2. 生成永久分享链接
3. 设置公众号关键词自动回复
4. 用户回复关键词自动获取工具链接
