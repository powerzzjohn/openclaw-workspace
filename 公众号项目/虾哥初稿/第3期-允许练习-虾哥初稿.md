# 一个简单的"允许练习"

## 当焦虑来临时，试着对自己说这句话

你有没有发现，当我们感到焦虑的时候，第一反应总是想要"做点什么"来消除它？但越是用力对抗，那种不舒服的感觉反而越强烈。

去年冬天，我经历了人生中最焦虑的一段时期。工作出问题，家里老人生病，孩子成绩下滑。所有事情好像约好了似的，同时压过来。

那天晚上，我躺在床上，脑子里像过电影一样，一遍遍回放各种糟糕的可能。越想越清醒，越清醒越焦虑。

我试过深呼吸，试过数羊，但就像用手压弹簧，压得越用力，反弹得越厉害。凌晨三点，我索性坐起来发呆。

就在那一刻，我想起了海灵格的话："允许一切，如其所是。"我试着对自己说："我允许我现在很焦虑。"

奇怪的是，当我不再试图"消灭"焦虑，而是允许它存在的时候，那种紧绷的感觉竟然开始松动了。

这个练习很简单。当焦虑来临时，停下来，深呼吸，然后真诚地说："我允许我现在感到焦虑。"

你会发现，当你不再和情绪对抗，它反而开始流动。就像被堵住的水流，不再用力堵它，它自然会找到出口。

允许不是放弃，是给力量腾出空间。当我们不再消耗能量对抗焦虑，这些能量就可以用来真正解决问题。

下次焦虑来临时，不妨试试。说三遍"我允许"，感受每一个字。看看你的身体有什么变化？
