# 始运工作助手 - 每日任务监控

## 触发条件
- 每天上午 9:00 自动运行
- 用户主动询问时

## 工作流程

### 1. 收集当日信息
```bash
# 提醒事项
remindctl today --json

# 日历事件
# 使用 macOS 自带 calendar 或通过其他方式获取

# 备忘录
memo notes
```

### 2. 分析维度
- **今日紧急**：今天到期的事项
- **未来3天**：近期需要关注的事项
- **未来1个月**：中长期规划

### 3. 输出格式
- 总结当前状态
- 识别冲突或风险
- 提供优化建议

## 手动触发
用户说"始运工作助手"或"查看今天的工作安排"时执行。
